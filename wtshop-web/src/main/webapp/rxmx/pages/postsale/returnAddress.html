退货地址在何处查询

前往“我的--我的订单”，找到对应订单后选择“申请售后”，填写退货说明及上传商品照片。

一、退货退款时效

1-3个工作日首次审核 → 审核通过后，短信通知 → 请按短信通知填写退货 → 仓库收到商品，1-7个工作日拆包验货 → 验货成功后，应退货款于1-7个工作日按订单付款方式原路退回。

二、运费填写方式

错发商品，运费商城承担，

注：

1、暂不支持到付件，亲先垫付，并保留好快递面单，客服后期退回相应金额；

2、若亲喜欢并愿意留下收到的商品，客服也会帮助提供退差价服务哦！
