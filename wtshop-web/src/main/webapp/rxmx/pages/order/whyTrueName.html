为何购物要实名制

一、原因

根据海关总署公告通知，若购买跨境商品不进行实名认证，或不提供收货人身份证照片校验，则商品无法发出。

部分商品海关要求需要进行收货人实名，并要求收货人提供身份证影印件进行校验，具体以下单的提示为准。

您的实名认证信息仅作清关用途，商城将严格保密。

二、操作步骤

电脑端点击“个人中心”-“我的实名认证”； 手机APP最新版找到我的，点击您的头像进入个人信息选择“我的实名认证”。

1.填写真实姓名( 必填 )

2.填写身份证号码( 必填 )

3.上传身份证正反面( 选填　建议上传 )

填写以上信息后，保存完成信息提交，即可下单购买商品啦~

温馨提示：

1.进行实名认证时，姓名必须为简体字，若有繁体字行为，则无法通过。

2.部分商品清关时，需提供收货人实名信息对身份证影印件进行校验。因此，购买该类商品时，必须上传身份证正反面进行认证，否则无法下单购买。

3.先付款后实名认证用户，可在“我的” → “我的实名认证” → “添加实名信息”中上传身份信息进行验证。

4.目前海关规定：个人购买跨境商品年度交易限值为人民币2万元，若超出限额，则订单会自动关闭，建议明年下单购买即可
