为何支付金额跟商品标价不同

当您遇到实际支付金额显示明显跟商品标价不符时，您可以核实是否以下三种情况：

1.运费

①商品基本满xx元包邮，不足xx元需承担x元运费，以商品活动后总金额为准，不含税费。

②特殊商品（如超大超重或者液体等商品）及入驻商家以页面显示为准。

2.活动商品

商品处于活动预告阶段，尚未正式开始活动。

3.税费情况

购买跨境商品，除支付商品正常价格外，还需依法缴纳综合税。

商品税费 = 完税价格×税率

完税价格( 实际支付价格 )　= 货物零售价格 + 运费 + 保险费

例：

若订单中包含A商品2件，单价100元，B商品2件，单价300元。A商品参加活动优惠了30元，B商品参加活动优惠了10元，使用优惠券后A商品又优惠了10元。商品A运费4元，商品B运费6元。A商品适用税率10%，B商品适用税率5%。

A商品的税费=完税价格*税率=（100*2+4-30）*10%=17.4 元

B商品的税费=完税价格*税率=（300*2+6-10）*5%=29.8 元

则订单需要缴纳的税费=A商品的税费+B商品的税费=17.4 + 29.8=47.2 元

若非以上几种情况，您可将商品界面截图，点击“帮助与客服”→“给客服留言”，具体描述问题，客服为您处理。
