开团流程

开团流程：

1.  点击团购按钮。

2. 提交并成功支付订单。

3. 查看订单-查看团详情-点击“邀请好友参团”按钮可将你发起的团购分享给好友。

4. 在团有效时间内，凑齐人数则拼团成功；未凑齐人数则拼团失败，系统将原路退款到付款账户。

参团流程：

1.  点击“去拼团”按钮进入拼团详情页，点击“我要参团”按钮。

2. 提交并支付订单。

3.  查看订单-查看团详情-点击“邀请好友参团”按钮可将你发起的团购分享给好友。

4.  在拼团有效时间内，凑齐人数则拼团成功；未凑齐人数则拼团失败，系统将原路退款到付款账户。

其他说明：

1.  在安吃商城APP—我的—我的拼团中，可以查看自己的拼团。

2. 团购商品支持7天无理由退换货，生鲜类商品除外（生鲜商品具有新鲜度、易腐蚀性等特点限制，目前不支持无理由退换货）。
