任性猫商城隐私政策

 

一.个人信息的收集

1.1

您已知悉且同意，在您注册任性猫商城用户帐号或使用任性猫商城提供的服务时，任性猫商城将记录您提供的相关个人信息，如：姓名、手机号码等，上述个人信息是您获得任性猫商城提供服务的基础。同时，基于优化用户体验之目的，任性猫商城会获取与提升任性猫商城服务有关的其他信息，例如当您访问任性猫商城时，我们可能会收集哪些服务的受欢迎程度、浏览器软件信息等以便优化我们的服务。

二.使用者非个人识别信息

2.1

2.1任性猫商城将从用户和网站访问者中记录非个人识别信息，包括但不限于IP地址、档案信息、用户总量数据和浏览器类型。这些数据将用来管理网站、追踪用途和改进网站服务。该等非个人识别信息可能会与第三方共享，以向用户们提供更多适当的服务和广告。用户的IP地址会被记录下来以供安全与监控之用。

三.个人信息的管理

3.1

为了向您提供更好的服务或产品，任性猫商城会在下述情形使用您的个人信息：

3.1.1

根据相关法律法规的要求；

3.1.2

根据您的授权；

3.1.3

根据任性猫商城相关服务条款、应用许可使用协议的约定。

3.2

此外，您已知悉并同意：在相关法律法规允许的范围内，任性猫商城可能会将您非隐私的个人信息用于市场营销，使用方式包括但不限于：在任性猫商城平台中向您展示或提供广告和促销资料，向您通告或推荐任性猫商城的服务或产品信息，以及其他此类根据您使用任性猫商城服务或产品的情况所认为您可能会感兴趣的信息。

3.3

未经您本人允许，任性猫商城不会向任何第三方披露您的个人信息，下列情形除外：

3.3.1

任性猫商城已经取得您或您监护人的授权；

3.3.2

司法机关或行政机关给予法定程序要求任性猫商城披露的；

3.3.3

任性猫商城为了保护公共会员和服务会员的安全；

3.3.4

任性猫商城为维护自身合法权益而向用户提起诉讼或仲裁时；

3.3.5

根据您与任性猫商城相关服务条款、应用许可使用协议的约定；

3.3.6

法律法规规定的其他情形。

3.4

除上述第3.3条规定属免责外，下列情况时任性猫商城亦毋需承担任何责任：

3.4.1

由于您将用户密码告知他人或与他人共享注册帐户，由此导致的任何个人资料泄露；

3.4.2

任何由于黑客攻击、计算机病毒侵入或发作、因政府管制而造成的暂时性关闭等影响网络正常经营之不可抗力而造成的个人资料泄露、丢失、被盗用或被篡改等；

3.4.3

由于与本网站链接的其它网站或其他任何第三方所造成之个人资料泄露及由此而导致的任何法律争议和后果。

四.个人信息的保护

4.1

任性猫商城将尽一切合理努力保护其获得的用户个人信息。为防止用户个人信息在意外的、未经授权的情况下被非法访问、复制、修改、传送、遗失、破坏、处理或使用，任性猫商城已经并将继续采取以下措施保护您的个人信息：

4.1.1

以适当的方式对用户的个人信息进行加密处理；

4.1.2

在适当的位置使用密码对用户个人信息进行保护；

4.1.3

限制对用户个人信息的访问；

4.1.4

其他的合理措施。

4.2

尽管已经采取了上述合理有效措施，并已经遵守了相关法律规定要求的标准，但任性猫商城仍然无法保证您的个人信息通过不安全途径进行交流时的安全性。因此，用户个人应采取积极措施保证个人信息的安全，如：定期修改帐号密码，不将自己的帐号密码等个人信息透露给他人。

4.3

您知悉：任性猫商城提供的个人信息保护措施仅适用于任性猫商城平台，一旦您离开任性猫商城，浏览或使用其他网站、服务及内容资源，任性猫商城即没有能力亦无义务保护您在任性猫商城以外的APP或网站提交的任何个人信息，无论您登陆或浏览上述APP或网站是否基于任性猫商城的链接或引导。

4.4

账户安全

4.4.1

五.个人信息的访问

5.1

当您完成任性猫商城的帐号注册后，您可以查阅或修改您提交给任性猫商城的个人信息。一般情况下，您可随时浏览、修改自己提交的信息，但出于安全性和身份识别的考虑，您可能无法修改注册时提供的某些初始注册信息及验证信息。

六.对token的使用

6.1

token可以帮助网站辨认注册用户，计算用户数量，通常被各网站用来判定完成注册的用户是否已经实现登录。任性猫商城承诺，对token信息的研究仅用于提升服务/产品质量及优化用户体验之目的。同时，如不希望个人信息保留在token中，您可以对浏览器进行配置：选择"拒绝token"或"当网站发送token时通知您"，您知道，鉴于任性猫商城的服务是通过支持token来是实现的，完成关闭token的操作后，可能影响到您访问任性猫商城或不能充分取得任性猫商城的服务。您不禁用token时，可能会得到提示：是否在下一次进入此网站时保留用户信息以便简化登录手续（如一键登录）。
